# ha-youversion-addon

Repositório de addons do Home Assistant com integração à **YouVersion Bible API**.

## Addons disponíveis

### 📖 YouVersion Versículo do Dia
Cria automaticamente o sensor `sensor.youversion_versiculo_do_dia` com o versículo do dia, texto completo e referência bíblica.

## Como instalar

Clique no botão abaixo para adicionar o repositório automaticamente:

[![Adicionar ao Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FSEU_USUARIO%2Fha-youversion-addon)

Ou manualmente:
1. No Home Assistant, vá em **Configurações → Addons → Loja de Addons**
2. Clique no menu (⋮) → **Repositórios**
3. Adicione a URL: `https://github.com/SEU_USUARIO/ha-youversion-addon`
4. Procure por **"YouVersion Versículo do Dia"** e instale
5. Configure sua **App Key** nas opções do addon e inicie

## Pré-requisitos

- Conta na [YouVersion Platform](https://platform.youversion.com)
- App Key obtida no portal de desenvolvedores
